#냠냠코치 (ProPro)

식단 분석을 기반으로 개인 맞춤형 건강 코칭 경험을 제공하는 서비스입니다. 음식별 영양 성분 데이터를 활용해 사용자가 작성한 식단을
직접 관리하고, 부족한 영양소나 맞춤 추천 메뉴를 안내합니다. 챌린지, 커뮤니티, 팔로우 기능 등 건강 관리에
도움이 되는 다양한 상호작용도 함께 제공합니다.

## 프로젝트 개요

- **목적**: 사용자들이 식단을 편하게 관리할 수 있게 도와주고, AI를 활용한 다양한 기능들을 제공합니다.
- **핵심 가치**: 기록의 편의성, 신뢰도 높은 영양 분석, 커뮤니티 기반 동기 부여를 중심으로 설계합니다.

## 주요 기능

- **식단 관리**: 식단 작성·조회·삭제 및 개인 식단 DB 저장.
- **회원 관리**: 회원 정보 작성·조회·삭제, 로그인/로그아웃 지원.
- **소셜 기능**: 팔로우, 챌린지 참여, 게시판을 통한 정보 교류.
- **이미지 기반 식단 인식**:
  - 업로드한 식단 사진에서 음식 종류 자동 파악.
  - 음식별 영양 성분 데이터 매핑 후 시각화.
  - 개인 식단 DB 자동 저장 및 부족 영양소 분석.
  - 부족 영양소를 바탕으로 맞춤 음식 추천.
- **운동 기록 관리**: 수행한 운동을 검색해 소모 칼로리를 계산·저장.
- **영양 분석 및 리포트**: 하루/주간 섭취 내역, 질환 정보, 다이어트 여부 등을 기반으로 AI가 부족 영양소를 분석하고 그래프와 함께 추천 제공합니다.

## 서비스 흐름

1. **기록**: 사용자가 식단을 텍스트 또는 이미지로 등록하면 음식별 영양 정보를 연동합니다.
2. **분석**: AI가 섭취 내역과 개인 목표(운동, 질병 관리 등)를 바탕으로 영양 균형을 진단합니다.
3. **추천**: 부족한 영양소를 채울 수 있는 음식과 운동을 제안하고, 챌린지 참여나 커뮤니티 교류로 동기를 부여합니다.
4. **리포트**: 그래프로 시각화된 하루/주간 리포트로 진행 상황을 확인합니다.

## 프로젝트 목표

- 사용자가 쉽게 식단을 기록하고 AI 기반 분석 결과를 통해 맞춤형 영양 코칭을 받을 수 있도록 지원합니다.
- 건강 목표 달성을 위한 챌린지와 커뮤니티 기능으로 사용자 참여도를 높입니다.
- 운동 기록과 칼로리 소비 데이터를 결합해 종합적인 건강 관리 경험을 제공합니다.

## 향후 계획

- 음식 영양 성분 데이터의 신뢰도 확보 및 최신화 전략 수립.
- 운동 칼로리 정보 제공을 위한 데이터 소스(DB 또는 외부 API) 확정.
- AI 분석 결과를 시각화하는 그래프 및 추천 모델 고도화.
