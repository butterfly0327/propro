식단 분석을 활용한 개인 맞춤형 건강 코칭 서비스(이하 “냠냠코치”라 한다.)를 
구현하려고 한다. 음식 별 영양성분 데이터를 활용하여 작성한 식단을 사용자가 원하는 목표(운동, 질병 
등)에 따라 분석하여 정보를 제공한다. 또한 건강 목표를 위한 챌린지, 식단 
리뷰 및 다양한 정보를 교류할 수 있는 커뮤니티 등 다양한 기능을 지원한다. 

기능

- 식단 CRUD (작성/조회/삭제)
- 회원 CRUD (작성/조회/삭제)
- 로그인/로그아웃
- 팔로우
- 챌린지
- 게시판
- 식단 사진을 올리면 그 사진을 ai가 분석하여
    - 음식 종류 파악
    - 음식 영양성분 데이터를 매핑해서 보여주기
    - 사용자별 개인 식단 관리 DB에 저장할 수 있도록 하는 기능
    - 부족한 영양소를 파악해서 음식을 추천
- 본인이 어떤 운동을 했는지 검색해서 소모 칼로리를 구하고(DB 내 정보 필요 or API??) 이를 저장 및 관리할 수 있는 기능
- 사용자 하루/**주간** 섭취 내역, 본인의 질환, 다이어트 여부 등의 정보를 바탕으로(AI API사용) 부족한 **영양소 파악(그래프 형식으로 보여주기)** 및 추천
